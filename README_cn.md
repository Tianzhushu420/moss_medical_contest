一、项目概述
 
本项目基于ROS2 Humble开发，适配OriginCar系列移动底盘，面向大学生智能医疗机器人竞赛设计，实现室内环境地图构建、自主导航、物料定点配送、激光雷达避障全套功能。
整套代码分为底层驱动包 origincar_base 与导航工作空间 ros2_ws ，覆盖小车底盘控制、激光雷达驱动、SLAM建图、路径规划、多点循环配送任务逻辑，可直接在实体小车部署运行。
 
二、仓库目录结构
 
1.  origincar_base 
小车底层基础驱动包，包含底盘电机控制、里程计解析、小车参数配置、基础启动launch文件，负责硬件底层通信与运动控制。
2.  ros2_ws  导航功能工作空间
 
-  map ：SLAM建图生成的导航地图资源（pgm地图图像+yaml参数文件）
-  src ：业务功能源码，包含导航节点、多点循环配送任务、激光雷达数据处理脚本
-  launch ：功能启动文件，支持单独启动底盘、建图、导航、自动配送任务
-  params ：导航、雷达、代价地图配置yaml参数
- 各类功能包：激光雷达驱动、栅格地图、粒子滤波定位、路径规划算法实现
 
三、核心功能
 
1. 硬件驱动：激光雷达数据接收、底盘速度闭环控制、里程计解算；
2. SLAM建图：实时扫描室内环境生成可复用导航地图；
3. 自主导航：AMCL粒子滤波定位、全局+局部路径规划、动态障碍物避障；
4. 医疗配送任务：多点位循环导航，自动抵达目标点位完成物料配送；
5. 可视化调试：配套话题查看、地图显示、定位观测功能。
 
四、编译与运行步骤
 
1. 进入工作空间
 
```bash
  
cd ros2_ws
```
 
2. 编译全部功能包
 
```bash
  
colcon build
 
``` 
3. 刷新环境变量

```bash
  
source install/setup.bash
```
 
4. 启动底盘+导航任务

```bash
  
ros2 launch racecar nav10_map.launch.py
 
```
