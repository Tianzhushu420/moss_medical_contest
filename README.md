1. Project Overview
 
This project is developed based on ROS2 Humble and compatible with OriginCar series mobile chassis. Designed for the university intelligent medical robot competition, it realizes a full set of functions including indoor environment mapping, autonomous navigation, fixed-point material delivery and LiDAR obstacle avoidance.
 
The complete code consists of the underlying driver package  origincar_base  and the navigation workspace  ros2_ws . It covers chassis control, LiDAR driver, SLAM mapping, path planning and multi-point cyclic delivery task logic, and can be directly deployed and run on physical robots.
 
2. Repository Directory Structure
 
1.  origincar_base 
The underlying basic driver package of the robot, including chassis motor control, odometer parsing, vehicle parameter configuration and basic launch files, responsible for hardware underlying communication and motion control.
2.  ros2_ws  Navigation Workspace
 
-  map : Navigation map resources generated by SLAM (pgm map images + yaml parameter files)
-  src : Business source code, including navigation nodes, multi-point cyclic delivery tasks and LiDAR data processing scripts
-  launch : Function launch files, supporting independent startup of chassis, mapping, navigation and automatic delivery tasks
-  params : Configuration yaml files for LiDAR, navigation and cost maps
- Various functional packages: LiDAR driver, grid map, AMCL particle filter localization and path planning algorithm implementation
 
3. Core Functions
 
1. Hardware driver: LiDAR data reception, closed-loop chassis speed control and odometer calculation;
2. SLAM mapping: Real-time scanning of indoor environment to generate reusable navigation maps;
3. Autonomous navigation: AMCL particle filter localization, global & local path planning, dynamic obstacle avoidance;
4. Medical delivery task: Cyclic navigation among multiple waypoints to reach target positions and complete material delivery automatically;
5. Visual debugging: Supporting topic monitoring, map visualization and position observation.
 
4. Build & Operation Instructions
 
1. Enter the workspace
 
```bash
  
cd ros2_ws
 
``` 
2. Compile all functional packages
 
```bash
  
colcon build
 
``` 
3. Refresh environment variables
 
```bash
  
source install/setup.bash
 
``` 
4. Launch chassis and navigation tasks
 
```bash
  
ros2 launch racecar nav10_map.launch.py
 
```
